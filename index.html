crux.land

crux.land is a free registry service meant for hosting small ( < 10kB) single deno scripts. All uploaded scripts are immutable and will not be changed nor deleted unless there is a legal reason or if it is found malicious.
You may also request a subdomain to be associated with a script.
To add a script you just need select a file and upload it with the interface on the right.
